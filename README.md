# 시티반점 - 중식 배달 주문 앱

배민 스타일의 모던한 중식당 배달 주문 웹 애플리케이션

## 🚀 주요 기능

- 📱 **모바일 최적화** - 반응형 디자인으로 모든 기기 지원
- 🛒 **장바구니** - 실시간 장바구니 관리
- ⭐ **포인트 시스템** - 주문 시 5% 포인트 적립
- 💳 **다양한 결제 수단** - 카드/현금/계좌이체
- 📦 **주문 관리** - 주문 내역 조회 및 추적
- 🎨 **세련된 UI** - 그라디언트와 애니메이션 효과

## 🎯 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- LocalStorage (데이터 저장)

## 📦 설치 및 실행

1. 저장소 클론
```bash
git clone https://github.com/YOUR_USERNAME/씨티반점.git
cd 씨티반점
```

2. 브라우저에서 `index.html` 열기

또는 간단한 서버 실행:
```bash
python -m http.server 8000
```
